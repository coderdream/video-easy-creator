================================================================================
                              快速开始
================================================================================

1. 基础使用:
   ClaudeApiClient apiClient = new ClaudeApiClient();
   ClaudeRequest request = new ClaudeRequest()
       .setModel(CdConstants.CLAUDE_DEFAULT_MODEL)
       .setMaxTokens(1024)
       .addUserMessage("Hello, Claude!");
   ClaudeResponse response = apiClient.sendMessage(request);
   System.out.println(response.getTextContent());

2. 工具调用:
   ToolRegistry toolRegistry = ToolExecutor.createDefaultRegistry();
   ClaudeToolExecutor toolExecutor = new ClaudeToolExecutor(apiClient, toolRegistry);
   String result = toolExecutor.executeWithTools("请列出当前目录中的所有文件");
   System.out.println(result);

3. 运行测试:
   mvn test -Dtest=Claude*Test
   mvn test -Dtest=ToolRegistry*
