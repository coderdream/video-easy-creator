## 关键决策

### 1. 模型选择

**决策**: 默认使用 Claude Sonnet 4.5，支持动态切换

**理由**:
- 性能和成本的最佳平衡
- 支持工具调用和思考模式
- 适合大多数应用场景

### 2. 工具调用实现

**决策**: 实现完整的工具调用循环，支持 5 个内置工具

**理由**:
- 提供自动化能力
- 支持文件操作和命令执行
- 可扩展的工具注册机制

### 3. 上下文管理

**决策**: 实现会话管理和上下文压缩

**理由**:
- 支持长对话
- 控制 Token 成本
- 提升用户体验

### 4. 应用场景

**决策**: 优先实现翻译和代码生成

**理由**:
- 与项目现有功能结合紧密
- 高价值应用场景
- 易于验证效果
