## 注意事项

1. **免费额度限制**: NVIDIA NIM 免费通道有速率限制（约 60 Requests/Minute）
2. **单线程调用**: 不要多线程高并发调用，避免触发 429 错误
3. **思考标签**: 响应中可能包含 `<think>...</think>` 标签，需要过滤
4. **超大上下文**: MiniMax M2.1 支持 200k+ tokens，适合整篇字幕翻译
