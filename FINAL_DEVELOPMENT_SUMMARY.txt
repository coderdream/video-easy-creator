## 项目概述

ClaudeCodeUtil 是一个功能完整的 Claude API 集成工具，为 video-easy-creator 项目提供高级 AI 功能支持。项目分为 4 个开发阶段，逐步构建从基础 API 通信到高级应用场景的完整功能体系。

---

## 开发阶段总结

### Phase 1: 基础框架 (已完成)

**目标**: 建立 Claude API 通信基础

**交付物**:
- `ClaudeMessage.java` - 消息对象 (支持用户/助手角色)
- `ClaudeResponse.java` - 响应对象 (支持文本/工具调用/思考块)
- `ClaudeRequest.java` - 请求构建器 (链式 API)
- `ClaudeApiClient.java` - API 客户端 (HTTP 通信)
- `CdConstants.java` - 常量配置 (15 个 Claude API 常量)

**测试覆盖**:
- ClaudeApiClientTest (8 个测试)
- ClaudeMessageTest (9 个测试)
- ClaudeResponseTest (13 个测试)
- 总计: 30 个测试用例

**关键特性**:
✅ 支持多个 Claude 模型 (Opus 4.5, Sonnet, Haiku)
✅ 扩展思考模式 (Extended Thinking)
✅ 努力程度配置 (Effort Levels)
✅ 代理支持
✅ 完整的错误处理

### Phase 2: 工具调用机制 (已完成)

**目标**: 实现 Claude Tool Use 功能

**交付物**:
- `ToolDefinition.java` - 工具定义 (灵活的输入模式)
- `ToolResult.java` - 工具执行结果
- `ToolRegistry.java` - 工具注册表 (管理和执行)
- `ToolExecutor.java` - 工具执行器 (5 个内置工具)
- `ClaudeToolExecutor.java` - 工具调用循环管理

**内置工具**:
1. `list_files` - 列出目录文件
2. `read_file` - 读取文件内容
3. `write_file` - 写入文件内容
4. `execute_command` - 执行系统命令 (白名单限制)
5. `grep` - 搜索文件内容 (正则表达式)

**测试覆盖**:

- ToolRegistryTest (11 个测试)

**关键特性**:

## 使用示例

### 基础 API 通信

```java
ClaudeApiClient apiClient = new ClaudeApiClient();

ClaudeRequest request = new ClaudeRequest()
    .setModel("claude-opus-4.5-20251101")
    .setMaxTokens(1024)
    .addUserMessage("Hello, Claude!");

ClaudeResponse response = apiClient.sendMessage(request);
String result = response.getTextContent();
```

### 工具调用

```java
ToolRegistry registry = ToolExecutor.createDefaultRegistry();
ClaudeToolExecutor executor = new ClaudeToolExecutor(apiClient, registry);

String result = executor.executeWithTools(
    "请列出当前目录中的所有文件"
);
```

### 会话管理

```java
ClaudeSession session = new ClaudeSession()
    .addMessage(ClaudeMessage.userMessage("Hello"))
    .addMessage(ClaudeMessage.assistantMessage("Hi there!"))
    .setContext("user_id", "123")
    .updateTokenUsage(100, 50);

List<ClaudeMessage> lastMessages = session.getLastMessages(5);
```

### 上下文压缩

```java
ContextCompressor compressor = new ContextCompressor()
    .setMaxMessages(20)
    .setCompressionRatio(0.7);

List<ClaudeMessage> compressed = compressor.compress(messages);
```

### 翻译

```java
ClaudeTranslator translator = new ClaudeTranslator(apiClient);

String result = translator.translateTechnical(
    "The API client uses HTTP/2 protocol",
    "Chinese"
);
```

### 代码生成

```java
ClaudeCodeGenerator generator = new ClaudeCodeGenerator(apiClient);

String code = generator.generateCode(
    "Create a function that calculates factorial",
    "Java"
);
```

## 常见问题

### Q: 如何处理 API 限流?
A: ClaudeApiClient 内置了重试机制和指数退避策略，自动处理 API 限流。

### Q: 如何自定义系统提示词?
A: 可以通过 ClaudeRequest 的 setSystemPrompt() 方法自定义系统提示词。

### Q: 如何扩展工具功能?
A: 可以通过 ToolRegistry 的 registerTool() 方法添加自定义工具。

### Q: 如何处理长对话?
A: 使用 ContextCompressor 进行智能消息压缩，保留关键信息。

### Q: 如何支持新的翻译场景?
A: 可以在 ClaudeTranslator 中添加新的 TranslationScenario 枚举值。
