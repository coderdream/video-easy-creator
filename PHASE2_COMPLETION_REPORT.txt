================================================================================
                              工具调用流程
================================================================================

用户输入
   ↓
发送请求 + 工具定义
   ↓
Claude 返回响应
   ↓
检查是否有 tool_use
   ├─ 是 → 执行工具 → 构建工具结果消息 → 继续循环
   └─ 否 → 返回最终答案

================================================================================
                              内置工具说明
================================================================================

1. list_files
   - 功能: 列出指定目录中的文件
   - 参数: path (目录路径)
   - 返回: 文件列表

2. read_file
   - 功能: 读取指定文件的内容
   - 参数: path (文件路径)
   - 返回: 文件内容

3. write_file
   - 功能: 写入内容到指定文件
   - 参数: path (文件路径), content (文件内容)
   - 返回: 操作结果

4. execute_command
   - 功能: 执行系统命令
   - 参数: command (要执行的命令)
   - 返回: 命令执行结果
   - 限制: 命令白名单限制

5. grep
   - 功能: 在文件中搜索匹配的行
   - 参数: path (文件路径), pattern (搜索模式)
   - 返回: 匹配的行
